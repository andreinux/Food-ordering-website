# Fast Food Ordering System

A responsive front-end fast food ordering system built with **HTML, CSS, and Vanilla JavaScript**.

This project simulates the ordering experience of a modern self-service kiosk. Users can browse menu categories, select food items, adjust quantities, review their order, and place an order through a realistic front-end workflow.

## Features

* Interactive multi-section user interface
* Browse menu by category
* Increase and decrease item quantities
* Automatically calculate subtotals and total price
* Dynamic checkout rendering using JavaScript
* Remove individual items from the order
* Cancel the entire order
* Empty checkout state handling
* Place Order prototype
* Dine-in / Take-out selection prototype
* Random order number generation
* Start New Order functionality
